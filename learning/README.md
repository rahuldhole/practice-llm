# 🗺️ AI & Deep Learning Learning Roadmaps

Welcome! This directory contains three distinct, curated roadmaps designed for different learning paths and career goals in Artificial Intelligence. Select the roadmap that aligns with your objectives:

---

## 🗺️ Choose Your Path

### 1. [💼 AI Practitioner & LLM Engineer Roadmap](roadmaps/1-ai-practitioner-roadmap.md)
* **Target Audience**: Software Engineers, Data Scientists, and Applied AI Engineers.
* **Goal**: Build production-grade LLM applications, fine-tune existing models (SFT, PEFT, DPO, GRPO), design evaluation pipelines, and deploy models efficiently using the standard industry stack (`transformers`, `vLLM`, `trl`, `axolotl`).
* **Focus**: Applied engineering, API integrations, and model optimization.

### 2. [🌌 Deep Learning Systems & Architectures Roadmap](roadmaps/2-dl-systems-roadmap.md)
* **Target Audience**: Core Deep Learning Engineers, Systems Builders, and Compiler/Hardware-bound Engineers.
* **Goal**: Understand the low-level mechanics of deep learning by building Autograd engines, custom neural libraries, and Transformers (including RoPE, GQA, MoE) from scratch.
* **Focus**: Low-level math, building from first principles, and system/memory bottleneck profiling.

### 3. [🔬 AI Research Lab Roadmap](roadmaps/3-ai-researcher-roadmap.md)
* **Target Audience**: Aspiring Research Scientists, PhD Candidates, and ML Infrastructure Researchers.
* **Goal**: Prepare for top-tier labs (DeepMind, Anthropic, OpenAI, FAIR) by mastering scientific methodology, paper reproduction, distributed training (FSDP, ZeRO), Triton/CUDA kernel writing, scaling laws, and evaluation science.
* **Focus**: Hypothesis-driven experiments, training stability, large-scale compute, and novel architectures.
